## Descripción del Proyecto
Aplicación desarrollada en Python para la gestión de información de países mediante el uso de archivos CSV como mecanismo de almacenamiento persistente. El sistema permite agregar, actualizar, buscar, filtrar y ordenar registros, además de generar estadísticas básicas sobre los datos almacenados.

## Funcionalidades
- Carga automática de datos desde archivo CSV.
- Alta de nuevos países.
- Actualización de información existente.
- Búsqueda de países por nombre.
- Filtrado de registros según distintos criterios.
- Ordenamiento de países.
- Generación de estadísticas.
- Guardado automático de cambios.

## Estructura del Proyecto
- `main.py`: contiene el programa principal y el menú de opciones.
- `archivos.py`: lectura y escritura de archivos CSV.
- `consultas.py`: búsqueda, filtrado y ordenamiento.
- `estadisticas.py`: generación de estadísticas.
- `carga_y_edicion_paises.py`: alta y actualización de países.
- `paises.csv`: almacena los datos utilizados por la aplicación.
- `README.md`: documentación general del proyecto.
- `Documentacion.pdf`: informe académico y técnico solicitado en la consigna.

## Requisitos
Python 3.10 o superior.

## Instrucciones de Ejecución

3. Ejecutar el comando : python main.py
